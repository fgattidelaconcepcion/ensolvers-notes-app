# Ensolvers Notes App

Simple full-stack notes application built for the Ensolvers interview challenge.

---

##  Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React + Vite + TypeScript + Tailwind |
| Backend | Node.js + Express + TypeScript |
| Database | SQLite |
| ORM | Prisma |
| Architecture | REST API + Service + Repository pattern |
